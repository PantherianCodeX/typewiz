# Archived Planning Documents

This directory contains historical planning and analysis documents from Typewiz development.

These documents served their purpose during active development phases and are preserved for reference:

- **STANDARDS_ENHANCEMENT_SUMMARY.md**: Summary of coding standards enhancements
- **TEST_REFORM.md**: Test infrastructure refactoring plan
- **TEST_REORGANIZATION_ANALYSIS.md**: Analysis of test reorganization efforts
- **typewiz_export.md**: Checklist for exporting Typewiz to a standalone repository

These are not current documentation and should not be used as implementation guides. Refer to the main [CONTRIBUTING.md](../../CONTRIBUTING.md) for current development practices.
